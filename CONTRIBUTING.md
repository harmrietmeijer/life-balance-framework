# Contributing to the Life Balance Framework

Thank you for your interest in making the Life Balance Framework better!

## How to contribute

### Suggest improvements

Open an issue with:
- New self-assessment questions (must be backed by research)
- Micro-action suggestions for any pillar
- Corrections or clarifications to the behavioral science references

### Add a translation

1. Fork this repository
2. Create a new folder under `translations/` with your language code (e.g., `translations/es/`)
3. Translate the README.md and self-assessment scorecard
4. Submit a pull request

### Share your story

We'd love to hear how the framework has helped you. Open an issue tagged `story` or reach out via [nuvo.coach](https://nuvo.coach).

## Guidelines

- Keep content evidence-based — cite sources where possible
- Maintain an encouraging, non-judgmental tone
- This is a methodology repo, not a code repo — no application code belongs here

## Code of Conduct

Be kind. Be constructive. We're all here to help people live more balanced lives.
